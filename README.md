## рџ“ђ DAX Measures (25 Total)

### Group 1 вЂ” Core KPIs
```dax
Total Sales ВЈ = SUM(FACT_Weekly_Sales[sales_value])
Total Sales Units = SUM(FACT_Weekly_Sales[sales_units])
ASP = DIVIDE([Total Sales ВЈ], [Total Sales Units], 0)
Returns Rate % = DIVIDE([Total Returns Units], [Gross Sales Units], 0)
Sell-Through % = DIVIDE([Total Sales Units], [Total Sales Units] + [Total Stock Units], 0)
```

### Group 2 вЂ” Last Year Comparisons
```dax
LY Sales ВЈ = CALCULATE([Total Sales ВЈ], SAMEPERIODLASTYEAR(DIM_Calendar[Date]))
Sales vs LY ВЈ = [Total Sales ВЈ] - [LY Sales ВЈ]
Sales vs LY % = DIVIDE([Sales vs LY ВЈ], [LY Sales ВЈ], 0)
```

### Group 3 вЂ” Rolling Trackers
```dax
Rolling 13 Week Sales ВЈ = CALCULATE([Total Sales ВЈ],
    DATESINPERIOD(DIM_Calendar[Date], LASTDATE(DIM_Calendar[Date]), -91, DAY))

Rolling 5 Week Sales ВЈ = CALCULATE([Total Sales ВЈ],
    DATESINPERIOD(DIM_Calendar[Date], LASTDATE(DIM_Calendar[Date]), -35, DAY))
```

### Group 4 вЂ” Stock & Replenishment
```dax
Weeks Cover = DIVIDE([Total Stock Units], DIVIDE([Total Sales Units], 5, 0), 0)
On Order Units = SUM(FACT_Weekly_Sales[on_order_units])
```

### Group 5 вЂ” Best Sellers Rankings
```dax
Sales Rank Overall = RANKX(ALL(FACT_Weekly_Sales[style_code]), [Total Sales ВЈ],, DESC, DENSE)
Top 10 Flag = IF([Sales Rank Overall] <= 10, "Top 10", "Other")
```
